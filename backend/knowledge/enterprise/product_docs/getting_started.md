# NimbusCRM — Product Documentation: Getting Started

## Overview
NimbusCRM is a cloud-based Customer Relationship Management platform designed for B2B companies.

## Core Modules

### 1. Pipeline Management
Track deals, contacts, and accounts in a unified view.
- Drag-and-drop pipeline stages
- Automated follow-up reminders
- Email activity logging

### 2. Reporting & Analytics
Build custom dashboards and export reports.
- 50+ pre-built report templates
- Custom report builder (drag-and-drop)
- Scheduled email reports
- Real-time dashboard refresh

### 3. AI Insights (Enterprise)
Machine learning powered predictions.
- Churn probability scoring
- Deal close prediction
- Account health monitoring
- Automated next-action suggestions

### 4. Workflow Automation
Automate repetitive tasks.
- Trigger-based automations (e.g., on deal stage change)
- Email sequences
- Task assignments
- CRM field updates

### 5. Integrations
Connect your existing tools.
- Native integrations: 25+ apps
- REST API for custom integrations
- Zapier support for 1000+ apps

## Mobile App
Available on iOS and Android.
- View pipeline and contacts
- Log calls and meetings
- Receive push notifications for important updates
- Note: Full dashboard and report creation requires desktop

## Common Onboarding Issues
- **"I can't see my reports"**: Check that your role has View Reports permission (Admin > Users > Roles)
- **"Data not importing"**: Ensure CSV format matches our template (help.nimbuscrm.com/import)
- **"Integrations not syncing"**: Re-authenticate the integration in Settings > Integrations
